Frontend Development

Dit is hoe ik Frontend Development heb behaald

De opdracht

Voor Frontend Development kregen we de opdracht om 2 pagina's van een bestaande website exact na te maken in de vormgeving, waarbij je code bestaat uit nette vanilla HTML, CSS en Javascript.
Hiervoor hebben we 5 weken de tijd voor gekregen. De belangrijkste voorwaarde is dat het semantisch correcte HTML is en dat je website Mobile first is. Daarna mag je de keuze maken om de website Responsive te maken of alleen je aandacht legt op de surface laag, zoals animaties. Bij beide keuzes moet je minimaal 1 regel Javascript toepassen. Verder moet je website teogankelijk zijn met een screenreader en moeten alle keuzes worden vastgelegd in je Product Biografie.

Het proces

Les 1: Oefenen met Typografie en kleuren met HTML en CSS en intake opdracht maken.
Les 2: Leren Flexboxen adhv de gegeven opdrachten.
Les 3: Leren positioneren en animeren van objecten met CSS
Les 4: Verschillende states aan je website toepassen en vormgeven van allerlei attributen.
Les 5: Verdiepen in het animeren van de content.
Les 6: Hier rondde ik de Product Biografie en zette de puntjes op de i.

Het eindproduct

Het is mij gelukt om de website exact hetzelfde vorm te geven als de echte NOS website, met leuke animaties die ik heb geleerd in de lessen. In de lessen van FED hebben we veel geoefend met animaties en positioneren, waarbij ik er veel van heb geleerd voor mijn website. In mijn website krijg je dus verschillende animaties die past bij de topic van een artikel op de NOS homepagina. Wat ik nog eventueel bij had toegevoegd was een aanmeld formulier voor het nieuwsbrief.

De resultaten

Voor FED heb ik een 7 gehaald, met als feedback: "Alhoewel er altijd meer diverse content erin kon, heb je wel voldaan aan de voorwaardes die verwacht werden van de studenten. Je heb leuke animaties bedacht en de website is heel gelijk aan de echte NOS website."
Ik heb ontzettend veel over HTML, CSS en Javascript geleerd. Met de kennis uit de gemaakte oefenopdrachten kan ik nu meer dan dat ik eerst dacht.
